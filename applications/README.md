# VRP Applications
These applications provide the core, adaptation independent, userfacing functionality of VRP. Adaptations will reference these implementations.


## Core Applications
- vrp-contacts: address book
- vrp-dialer: dialer based on ModemManager
- vrp-log: call log
- vrp-messages: SMS and MMS based on ModemManager and mmsd-tng
- vrp-pim: calender with appointments and alarms
- vrp-cell-info: provides an easy interface for retrieving network info from ModemManager
- vrp-input: T9 backend for interface builders
- vrp-feedback: force feedback backend

## Extended Applications
- vrp-fm: file manager
