# VRP Adaptations
An "adaptation" refers to any hardware or software developed for a VRP based device. Someone may decide to utlizie the VRP stack to build a device and the final product, which includes the hardware desined and software written (drivers, interface, etc.), is considered the adaptation. The term "device target" may also be used to refer to an adaptation.
